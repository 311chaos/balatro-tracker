## Props

| Prop       | Type                                                                               | Default   | Description                          |
| ---------- | ---------------------------------------------------------------------------------- | --------- | ------------------------------------ |
| `variant`  | `default` \| `outline` \| `secondary` \| `ghost` \| `destructive` \| `link`        | `default` | Controls visual style                |
| `size`     | `xs` \| `sm` \| `default` \| `lg` \| `icon` \| `icon-sm` \| `icon-lg` \| `icon-xs` | `default` | Controls height and padding          |
| `disabled` | `boolean`                                                                          | `false`   | Disables interaction, reduces opacity |

## Usage

```tsx
import { Button } from '@/components/ui/Button';

<Button>Save</Button>
<Button variant="outline">Cancel</Button>
<Button variant="destructive">Delete</Button>
<Button variant="secondary" size="sm">Secondary</Button>
<Button render={<a href="/jokers" />}>Browse jokers</Button>
```
