# Dialog

Modal overlay built on Base UI's `Dialog` primitive. Renders into a portal with a backdrop blur and entrance/exit animations. The close button in the top-right corner is opt-out via `showCloseButton={false}` on `DialogContent`.

## Anatomy

```tsx
<Dialog>
  <DialogTrigger render={<Button />}>Open</DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Title</DialogTitle>
      <DialogDescription>Description</DialogDescription>
    </DialogHeader>

    {/* body content */}

    <DialogFooter showCloseButton>
      <Button>Confirm</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

## Props

### DialogContent

| Prop              | Type      | Default | Description                                     |
| ----------------- | --------- | ------- | ----------------------------------------------- |
| `showCloseButton` | `boolean` | `true`  | Show the X button in the top-right corner       |
| `className`       | `string`  | —       | Forwarded to the popup panel for custom theming |

### DialogFooter

| Prop              | Type      | Default | Description                                    |
| ----------------- | --------- | ------- | ---------------------------------------------- |
| `showCloseButton` | `boolean` | `false` | Append a "Close" dismiss button after children |
